# Live Screen Share

A screen sharing application project.

## Files

- `chat_screenshare.py` - Local screen viewer
- `client.py` - Client for streaming screen
- `server.py` - Server for receiving streams
- `network_protocol.py` - Network communication functions
- `config.py` - Configuration settings

## Setup

```bash
pip install mss numpy opencv-python pillow
```

## Usage

Start building your screen sharing application by adding functionality to each file.
